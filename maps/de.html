Municipality Emails

Email Providers of German Municipalities

What is this?

A map of all ~11,100 German municipalities showing which provider handles their official email, grouped by jurisdiction, based on public DNS records and other public network signals.

Context

Digital sovereignty: US-based providers are subject to the US CLOUD Act, which allows US authorities to request stored data, regardless of where it is physically hosted.

How does it work?

Each municipality's official domain is checked via DNS records, SMTP banners, ASN lookups, and other signals, then classified by provider type with confidence scoring.

Open source & open data

The code and data are on GitHub.
